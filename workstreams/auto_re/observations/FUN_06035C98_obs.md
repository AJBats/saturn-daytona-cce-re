## Computation Pattern

1. Read steering (+0x14) and velocity components (+0x64, +0x68)
2. Add steering to velocity, compare with threshold
3. Call sin function (FUN_06047D20) on intermediate values
4. Multiply results by 0x0002C000 (~0.171 in fixed-point)
5. Conditional path based on comparison result — either:
   a. Skip to simplified output, or
   b. Call cos (via FUN_06047D3C), sqrt, do more trig
6. Uses constant r7=0x4000 (16384) — likely a half-circle angle offset

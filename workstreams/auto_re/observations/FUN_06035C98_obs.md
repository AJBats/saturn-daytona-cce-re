## Computation Pattern

1. Read +0x14 (16-bit signed) and +0x64, +0x68 (32-bit values)
2. Add +0x14 to +0x64, compare sum with threshold
3. Call FUN_06047D20 (sin) on intermediate values
4. Multiply results by 0x0002C000 (~0.171 as fixed-point)
5. Conditional path based on comparison result — either:
   a. Skip to simplified output, or
   b. Call FUN_06047D3C (cos), FUN_0604818C (sqrt), continue with more computation
6. Uses constant r7=0x4000 (16384) as offset added before trig calls
